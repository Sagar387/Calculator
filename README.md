# Basic Calculator App

## Overview
This is a simple calculator web application that performs basic arithmetic operations such as addition, subtraction, multiplication, and division. It is built using **HTML, CSS, and JavaScript**.

## Features
- Addition, subtraction, multiplication, and division
- Clear button to reset calculations
- Responsive and user-friendly interface

## File Structure
```
calculator-app/
│── index.html       # Main HTML file (Calculator UI)
│── style.css        # Stylesheet for calculator design
│── script.js        # JavaScript logic for calculations
│── README.md        # Project documentation
```

## How to Use
1. Open `index.html` in a web browser.
2. Use the number buttons to input values.
3. Click on the arithmetic operator (+, -, *, /) and enter the second number.
4. Press `=` to compute the result.
5. Click `C` to clear the input and start over.

## Installation
No installation is required. Simply open `index.html` in any modern web browser.
